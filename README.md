# Link Archive
--------------
My DevOps learning project with app generated by ChatGPT

## Server setup
---------------
Ubuntu 24

Update server
```
sudo apt update && sudo apt upgrade
```

Restart system
```
sudo reboot
```

### Get Repo
------------
Get repo
```
sudo git clone https://github.com/Cryobs/link-archive.git -d /srv/link-archive
``` 

### SSH
-------
Install OpenSSH Server
```
sudo apt install openssh-server
```

Enable and start ssh service
```
sudo systemctl enable ssh
sudo systemctl start ssh
```
(you can login for a now)
